Prework Study Guide

HTML

  • the head element contains information about the webpage, only show to the browser and other developers.
  • The body element represents the visible content shown to the user.
  • Attributes are always defined at the beginning element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS refers to styling an element directly in an HTML file.
  • Internal CSS contains rules for the HTML file in the 'head' section.
  • External CSS is a seperate file containing CSS rules and linked in the 'head' section of an HTML file.

Git

  • git is a version control software, utilizing special git commands to perform tasks.
  • git status: checks what branch we are currently on.
  • git checkout -b "feature/branch-name": creates a new branch and switches to it.
  • git checkout "branch name": switches to the branch you searched.
